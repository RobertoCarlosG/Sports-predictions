
## REGLAS IRROMPIBLES DE DISEÑO

Permitido
✓Probabilidad como barra + porcentaje — nunca solo número flotante
✓Máx. 2 acciones visibles por pantalla sin scroll
✓Filtros avanzados solo detrás de botón explícito "Filtros"
✓Selector de deporte como pestañas horizontales con scroll
✓Fecha como chip seleccionable (Hoy · Mañana · Esta semana)
✓Acordeones colapsados por defecto en detalles técnicos
✓Estado del partido como badge de color (En vivo · Próximo · Final)

Prohibido
✗Inputs de fecha de texto — siempre selectores visuales
✗Más de 3 inputs visibles en cualquier pantalla sin interacción previa
✗Terminología técnica expuesta (game_pk, fetch_details, sync, boxscore)
✗Mostrar JSON o estructuras de datos crudas al usuario
✗Errores de API crudos — siempre mensaje amigable + reintentar
✗Tablas con más de 5 columnas visibles sin scroll horizontal explícito
✗Iconos sin etiqueta de texto en acciones primarias
